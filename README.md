# Exam Question Paper Analyzer (MVP)

This web app analyzes multiple exam question paper PDFs
and finds repeated / high-frequency questions.

## Features
- Upload 2 or more exam PDFs
- Detect repeated / similar questions
- Works for any English-language exam paper
- Simple UI (student-friendly)

## How to Use
1. Open the app link
2. Upload 2 or more question paper PDFs
3. Click Analyze
4. View repeated questions instantly

## Tech Stack
- Python
- Streamlit
- PDF Text Extraction
